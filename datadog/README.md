# Datadog Observability Lab

## Overview

This lab demonstrates onboarding infrastructure into Datadog, validating telemetry collection, analyzing host metrics, and building foundational observability workflows.

## Lab Components

### Host Onboarding

* Datadog account setup
* Agent deployment
* Host registration
* Connectivity validation

Reference:

* host-onboarding.md

### Windows Agent Installation

* Agent installation
* Service verification
* Telemetry validation

Reference:

* windows-agent-install.md

### Host Metrics Dashboard

* CPU monitoring
* Memory monitoring
* Disk monitoring
* Network monitoring

Reference:

* host-metrics-dashboard.md
